# Retail-Sales-Analytics-Dashboard-Excel-Data-Cleaning-Insights-
This project focuses on analyzing an Indian e-commerce retail dataset to uncover sales trends, customer behavior, and product performance. The goal was to transform raw, unstructured data into a clean, interactive dashboard that supports business decision-making.

🎯 Problem Statement 
Businesses often struggle to track performance across multiple sales channels, categories, and time periods.
This project solves that by creating a centralized dashboard that provides clear, actionable insights into sales performance.

🛠️ Tools & Skills Used
Microsoft Excel
Data Cleaning (removing duplicates, handling missing values)
Data Processing & Transformation
Pivot Tables & Pivot Charts
Slicers (Month, Category, Channel)
Dashboard Design & Data Visualization

📊 Key Features of Dashboard
Monthly sales performance tracking
Category-wise revenue analysis
Sales channel comparison (e.g., online platforms)
Interactive slicers for dynamic filtering
KPI metrics (Total Sales, Profit, Quantity)
